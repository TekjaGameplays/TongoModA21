

*ENG*
Download, Unzip, and Drag-n-Drop all to (\7 Days To Die\Mods) (or your Mods folder). 
If you have never used mods, create the "/Mods" folder inside the folder containing the game.
Start the game normally, the changes are applied automatically.

---
This method is valid for A21.2 and lower, it is possible that in later versions it will be done in another way.
---




*ESP*
Descarga, descomprime y arrastra todo a (\7 Days To Die\Mods) (o la carpeta donde tengas los Mods). 
Si nunca has utilizado mods, crea  la carpeta "/Mods" dentro de la carpeta que contiene el juego.
Arranca el juego normalmente, los cambios se aplican automáticamente.

---
Este método es válido para A21.2 e inferiores, es posible que en posteriores versiones se realice de otra forma.
---
